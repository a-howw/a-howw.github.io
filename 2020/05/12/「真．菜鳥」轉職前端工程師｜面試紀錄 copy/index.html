「真．菜鳥」轉職前端工程師｜面試紀錄

2020-05-12

會想寫這篇文是因為，上網看了一下感覺像我這種真．菜鳥，應該蠻多人都跟我一樣不太敢寫，所以硬著頭皮上了 XD
我高中大學都是設計系，目前出社會二年，非程式相關工作經驗一年多，
雖然已經找到工作也過了試用期才發這篇文，算是一個給自己的紀錄也好~

投遞歷程

約 2019 11月初開始投履歷，我先投 104 上的百人大公司，至於為什麼，可以說是想看大小公司面試的差異吧
104 上有半公開履歷，不公開是因為騷擾電話夠多了⋯主動詢問的還是有幾家，但通常都不是很理想，所以皆並無進一步面試，後來改投 CakeResume，但只投了 7 家公司

投遞結果參考——

104 投遞

數量/已讀/面試：29/22/7 (後有婉拒兩家面試)，分別如下

  • 威盛電子
  • 燦坤
  • 泰偉電子 (婉拒)
  • 誠品
  • 精誠資訊
  • 易遊網
  • 露天 (婉拒)

104 主動通知

數量/通知前端職缺的/面試：26/8/0

CakeResume 投遞

數量/已讀/面試：6/5/3，分別如下

  • 伽博資訊有限公司
  • 台灣宸運雲端有限公司
  • 達易智造有限公司
    從面試到最後錄取，總共一個月，沒想到北部真的找工作頗快

以下依照面試順序開始面試紀錄——

威盛電子股份有限公司

人生第一次電話面試，面試下來感覺蠻不錯的，面試官就是把你當同事想認識你這樣，開場先請我自我介紹，後主管再介紹開出這個職缺的原因、公司單位、目前部門同仁組成、部門負責產品項目、協作流程等等。

技術方面問答：

  • 你目前使用的編輯器？
  • 你會使用 Git 嗎？
  • 對於原生 JS 是否有研究？
  • Vue.js、React、Angular三大框架是否有了解過？

測驗：無

面試方式：電話面試

面試時長：約 10–15分鐘

結果：無聲卡

燦坤股份有限公司

部門組長先面試，表示因為高層換人等等原因，部門內部正在改組，網頁的部分需要改版才開出這個職缺。
工作內容大致上為電商平台開發、維護，且節慶活動時段會比較忙，需要加班，可以申請加班費。
再來換部門主管進來稍微談一下，後很快速地請人資說明公司福利制度的部分。人資針對履歷上的各種小細節挑問題問，針對履歷的部分也給了一些建議。

面試官發問問題——

技術方面問答：無

部門組長：

．對這一份工作有什麼期待？
．有沒有看過公司的網站？有什麼想法？

部門主管：

．工作時什麼事情會覺得開心？
．什麼事情會覺得不開心、有壓力的？

人資：

．離開前一份工作原因？
．前公司經營項目？
．大概什麼時候開始找工作？
．目前面試了幾間公司？
．目前投遞的公司有鎖定在哪些產業類別嗎？
．過去跟你相處的人通常會怎麼形容你？
．未來三年對自己的期許？
．你覺得你應徵這份工作有哪些強項是有加分作用的？
．除了專業技術面，你認為這份工作需要什麼樣的特質能力？你認為你是這樣的人嗎？
．為什麼會想應徵我們公司？
．對我們公司的印象是什麼？
．以做為消費者角度來看，目前公司的電商介面有沒有讓你覺得不錯的地方？有什麼地方是需要再加強的地方？

測驗：無

面試方式：面對面面試；第一關部門組長，第二關部門主管，第三關人資

面試流程：

  1. 部門組長面試：自我介紹 → 部門組長發問 → 部門組長介紹公司、工作內容 → 問題提問
  2. 部門主管面試：部門主管發問 → 問題提問
  3. 人資面試：自我介紹 → 人資發問 → 問題提問

面試時長：約1小時

結果：備胎一枚（無聲卡了一個月，才突然來電）

誠品股份有限公司

主要工作內容為電商平台重構、開發、切版等等，專案目標則為使消費者於電商購買產品時，有更佳的使用體驗、更容易買到想要的東西。
工作上會跟 PD、視覺設計、UI/UX 及後端配合，過程中若有新的使用者需求，經討論後切分需求再分配工作。也表示希望以有效的方式處理工作，盡量避免加班。
溝通工具主要使用 Slack，若事情真的很急則當面討論。
考核部分，試用期考核是希望在短期間內幫助了解工作內容及工作狀況，正式同仁一年會有兩次考核時間，年中跟年尾，考核結果是調薪及升遷依據。
過程中有一些閒聊，可以知道部門主管跟部門同仁之間的相處算輕鬆愉快。

面試官發問問題 — —

技術方面問答：針對應徵作品內使用技術做提問

部門主管：

．通常都是在職進修，為什麼選擇離職進修？
．自學這段時間的流程、使用的資源及如何自訂學習目標？
．在學習過程中遇到最困難的事情？如何解決？
．未來三到五年對自己的期許？

部門同仁：

．合作同事有幾個？工作如何配合？
．應徵作品完全為課程教學內容，還是有自己撰寫的部分？

人資：

．前一份工作遇到最有成就感的事？
．上一份工作待遇？
．什麼情況下會覺得有壓力？如何紓解壓力？
．在團隊中扮演什麼樣的角色？
．什麼樣的人是比較沒有辦法忍受的？
．對下一份工作有哪些期待是很重視的？
．希望報到時間？
．目前為止有面試其他公司的前端嗎？
．還有安排其他面試嗎？

前端框架：Vue.js

測驗：無

面試方式：面對面面試；部門主管、部門同仁、人資，三人同時一起面試

面試流程：自我介紹 → 面試官發問 → 面試官介紹公司、工作內容 → 問題提問

面試時長：約30分鐘

結果：備胎一枚（無聲卡了一個月，才突然來電）

精誠資訊股份有限公司

測驗完先來面試的是人資，針對履歷問問題，後表示這個職缺開出來的原因是剛成立的團隊缺人，該團隊主要是負責研發新產品，且因為會參展，需要幫忙展區設計，工作上配合的同仁會有 PM、UI/UX、後端。
接著部門主管面試，部門主管負責產品專案管理，主要工作內容是研發產品，產品上線之後的工作內容以維護及增加新功能為主，有可能會需要支援其他部門。
也表明老闆喜歡改東改西，後詢問如果有要做一些平面文宣設計是否願意協助，以及若有參展是否願意支援。中途請部門負責 UI/UX 的同仁過來稍微解說一下工作內容及開發流程。
最後稍微參觀公司大樓的員工休息區，環境算是很舒適。

面試官發問問題——

技術方面問答：無

部門主管：

．前端能力如何？
．前一份工作待遇？
．最快可以什麼時候開始上班？

部門同仁：

．針對目前開發中的產品你會使用什麼技術？
．有做過哪些比較特殊的作品？

人資：

．為什麼離開前一份工作？
．前一份工作跟幾位同事合作及負責工作內容？
．前一份工作待遇？
．如何跟前一份工作的主管提離職？
．前一份工作需要面對客戶嗎？
．前一份工作最大的挫折？
．過去的主管如何形容你？
．未來職涯規劃？
．什麼時候開始找工作的？
．目前面試過什麼樣類型的公司？

測驗：圖形邏輯測驗、性向測驗（共1小時）

面試方式：面對面面試；第一關人資，第二關部門主管

面試流程：

  1. 測驗
  2. 人資面試：自我介紹 → 人資發問 → 人資介紹公司 → 問題提問
  3. 部門主管面試：自我介紹 → 部門主管介紹公司、工作內容及公司同仁 →問題提問

    面試時長：約1小時30分

    結果：錄取（約一週後通知）

易遊網股份有限公司

測驗完後部門主管面試，開頭請我先自我介紹後開始提問，且介紹部門組別的負責內容。
工作流程會由業務提出需求，然後再由主管依照程度分配工作及決定使用技術，主要工作是根據需求開發功能。
部門風氣較講求合作，專業分工細、各司其職，同事大致上相處融洽，也會互相幫忙。溝通工具使用 Slack，若是很急的話會直接當面溝通。
過程中也有一些閒聊，大致上過程愉快，有告知錄取結果約多久通知，也如期通知結果。

面試官發問問題——

技術方面問答：無

部門主管：

．自學使用的資源？
．應徵的作品使用什麼框架、開發約多久？
．寫程式過程中遇到問題如何解決？
．Vue.js 從完全不會到有點熟悉大約多久？
．在寫 Vue.js 時遇到的最大問題是什麼、如何解決？
．jQuery 熟嗎？
．寫測驗時有什麼感覺？
．對這一份工作有什麼憧憬？
．前一份工作團隊有幾人、合作及工作分配模式？
．前一份工作待遇？
．還有其他面試？
．投遞履歷時考量的項目？

測驗：前端基礎測驗（1小時），內容涵蓋 HTML & CSS, JavaScript, jQuery

面試方式：面對面面試

面試流程：

  1. 測驗
  2. 自我介紹 → 部門主管介紹公司、工作內容 → 問題提問

    面試時長：約1小時

    結果：感謝函（約一週後通知）
